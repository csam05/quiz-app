# Quiz App

An interactive and responsive quiz application built using HTML, CSS, and JavaScript.

## Features
- Responsive user interface
- Start quiz functionality
- Multiple-choice questions
- Instant answer validation
- Correct answers highlighted in green
- Incorrect answers highlighted in red
- Score calculation system
- Final result with motivational quote

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. Click the "Start Quiz" button
2. Questions appear one by one with four options
3. Select an answer
4. Correct answers turn green
5. Incorrect answers turn red
6. Final score is displayed after completing all questions
7. A quote is shown based on the score achieved

This project was created to practice:
